Fonction de chiffrement javascript :

Entrée :
	chars : chaine de caractères uniques, 
			la sortie est composé de ces caractères uniquement ainsi que de chiffres
	key : clé de chiffrement supplémentaire
	input : texte à chiffrer
	
Cette fonction peut chiffrer n'importe quel caractère ( chinois, symboles ... )
Elle est basé sur l'utilisation de char code
Maitrise des caractères de sortie

Démo : http://dginio.fr/crypt
